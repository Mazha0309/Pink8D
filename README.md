# Pink8D (P8D) v2.0

> A morphological programming language with thrust-based syntax and instant eject mechanism.

Pink8D 是一门基于**形态学（Morphology）**和**物理推力（Thrust）**的编程语言。所有语法都围绕 `8====D` 和 `8====>` 这两种视觉形态展开。**杆长即推力**，通过 `=` 的数量来决定运算强度。

## 🎯 项目特色

- **即时喷射** - `~` 后缀实现"边动边喷"，极大提升代码密度
- **形态美学** - 代码本身具有强烈的视觉冲击力
- **非对称设计** - 赋值封闭（`8...D`），位移开放（`8...>`）
- **四套执行引擎** - 解释器、字节码编译器+虚拟机、原生 C 代码转译器
- **65536 内存** - 比 v1.x 大 2 倍的内存池

## 📦 快速开始

### 环境要求

- Python 3.6+
- GCC 或 Clang 编译器（仅在使用 p8dforce.py 时需要）

### 安装

```bash
git clone https://github.com/your-repo/Pink8D.git
cd Pink8D
```

### 运行示例

```bash
# 使用虚拟机（推荐方式）
python3 src/p8cc.py examples/BG5CRL.8d    # 编译为字节码
python3 src/p8vm.py examples/BG5CRL.ejac  # 运行字节码

# 使用原生编译器（最高性能）
python3 src/p8dforce.py examples/pi.8d
./examples/pi
```

## 🛠️ 工具链详解

### 1. p8cc.py - 字节码编译器

将 `.8d` 源码编译为 `.ejac` 字节码文件。

```bash
python3 src/p8cc.py <source.8d>
# 输出：<source>.ejac
```

**编译特性**：
- 正则词法分析：`r'8[=.]*D~?|8[=]*>~?|<[=]*8~?|8\{|\}D|~'`
- 字节码格式：`opcode | 0x80`（高位置喷射标记）
- 内存池：65,536 字节

### 2. p8vm.py - 专用虚拟机

运行由 `p8cc.py` 生成的 `.ejac` 字节码文件。

```bash
python3 src/p8vm.py <source.ejac>
```

**执行特性**：
- 即时喷射：`putchar()` 与主操作并行执行
- 环形缓冲区：指针越界自动环绕
- 性能：比解释器快 2-3 倍

### 3. p8dforce.py - 原生转译编译器 ⭐

将 `.8d` 代码转译为优化的 C 代码，并通过 GCC/Clang 编译为原生二进制。

```bash
python3 src/p8dforce.py <source.8d>
# 自动生成并执行：<source> (Linux/macOS) 或 <source>.exe (Windows)
```

**性能优化**：
- `-O3` 编译优化
- 1MB 内存池
- 自动选择 clang/gcc
- 自动清理中间文件

## 🔧 核心指令

### 赋值指令

| 语法 | 公式 | 示例 |
|------|------|------|
| **压力赋值** | `8====D` | `m[ptr] += n` |
| **减压赋值** | `8....D` | `m[ptr] -= n` |

**公式**：`n = '=' 数量 - '.' 数量`

### 位移指令

| 语法 | 公式 | 示例 |
|------|------|------|
| **右向位移** | `8===>` | `ptr += 4` |
| **左向位移** | `<===8` | `ptr -= 4` |

**公式**：`n = '=' 数量 + 1`

### 逻辑指令

| 语法 | 语义 |
|------|------|
| `~` | 输入：`m[ptr] = getchar()` |
| `8D~` | 喷射：`putchar(m[ptr])` |
| `8{` | 循环：`while(m[ptr])` |
| `}D` | 结束：`}` |

### 即时喷射

任何指令加 `~` 后缀都会在动作完成后立即喷射：

```pink8d
8==D~   // 加 2 → 喷射
8>~     // 右移 1 → 喷射
<8~     // 左移 1 → 喷射
```

## 🛠️ 开发指南

### 词法分析正则

```python
pattern = r'8[=.]*D~?|8[=]*>~?|<[=]*8~?|8\{|\}D|~'
```

### 字节码格式

```python
OP_MAP = {
    "8{":    0x01,  # 循环开始
    "}D":    0x02,  # 循环结束
    "~":     0x03,  # 输入
    "ADD":   0x04,  # 加法
    "SUB":   0x05,  # 减法
    "MOV_R": 0x06,  # 右移
    "MOV_L": 0x07   # 左移
}

EJECT_BIT = 0x80  # 喷射标记
```
