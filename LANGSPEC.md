## 1. 设计哲学

### 1.1 核心原则

- **形态即功能**：代码的视觉长短直接对应操作的物理强度
- **非对称美学**：赋值指令是封闭的（`8...D`），位移指令是开放的（`8...>`）
- **即时性**：通过 `~` 后缀实现"边动边喷"

### 1.2 物理类比

```
赋值：   8====D     → 向内挤压，数值增加（推力注入）
位移：   8====>     → 向右喷射能量，带动指针
循环：   8{ ... }D  → 逻辑空腔，能量循环
```

## 2. 词法规范

### 2.1 正则表达式

```python
pattern = r'8[=.]*D~?|8[=]*>~?|<[=]*8~?|8\{|\}D|~'
```

### 2.2 Token 类型

| Token 模式 | 类型 | 示例 | 公式 |
|-----------|------|------|------|
| `8[=.]*D~?` | 赋值指令 | `8====D`, `8....D~` | 值 = `=` 数量 - `.` 数量 |
| `8[=]*>~?` | 右移指令 | `8>`, `8===>~` | 距离 = `=` 数量 + 1 |
| `<[=]*8~?` | 左移指令 | `<8`, `<===8~` | 距离 = `=` 数量 + 1 |
| `8{` | 循环开始 | `8{` | while(m[ptr]) |
| `}D` | 循环结束 | `}D` | } |
| `~` | 输入 | `~` | m[ptr] = getchar() |

### 2.3 词法分析示例

```python
source = "8====D~ ===>8{ ~ }D"
tokens = re.findall(pattern, source)
# ['8====D~', '===>8', '8{', ' ', '~', ' ', '}D']
# 实际 token: ['8====D~', '8>', '8{', '~', '}D']
```

## 3. 指令集

### 3.1 基础指令

| 语法 | 名称 | C 等价 | 视觉含义 |
|------|------|--------|----------|
| `8====D` | 压力赋值 | `m[ptr] += n` | 向内挤压 |
| `8....D` | 减压赋值 | `m[ptr] -= n` | 向外抽吸 |
| `8>` | 右向位移 | `ptr += 1` | 动力右喷 |
| `8===>` | 右向位移 | `ptr += 4` | 动力右喷 |
| `<8` | 左向位移 | `ptr -= 1` | 动力左喷 |
| `<===8` | 左向位移 | `ptr -= 4` | 动力左喷 |
| `~` | 流体吸入 | `m[ptr] = getchar()` | 从环境吸入 |
| `8{` | 循环开启 | `while(m[ptr]) {` | 开启逻辑空腔 |
| `}D` | 循环结束 | `}` | 封闭空腔 |

### 3.2 位移量公式

```
位移量 = 等号数量 + 1

示例：
  8>     → 0 + 1 = 1
  8=>    → 1 + 1 = 2
  8==>   → 2 + 1 = 3
  8===>  → 3 + 1 = 4
```

### 3.3 赋值量公式

```
赋值量 = 等号数量 - 点号数量

示例：
  8D     → 0 - 0 = 0
  8=D    → 1 - 0 = 1
  8....D → 0 - 4 = -4
  8==D.  → 2 - 1 = 1
```

## 4. 即时喷射机制

### 4.1 概念

**即时喷射（Instant Eject）**：任何改变状态的指令加上 `~` 后缀，都会在动作完成后立即执行 `putchar(m[ptr])`。

### 4.2 喷射指令表

| 复合语法 | 动作序列 | 说明 |
|---------|----------|------|
| `8====D~` | 加压 n → 喷射 | 修改数值后立即输出 |
| `8....D~` | 减压 n → 喷射 | 递减数值并输出 |
| `8>~` | 右移 1 → 喷射 | 移动到新格并输出 |
| `8===>~` | 右移 4 → 喷射 | 移动到新格并输出 |
| `<8~` | 左移 1 → 喷射 | 移动到新格并输出 |
| `<===8~` | 左移 4 → 喷射 | 移动到新格并输出 |

### 4.3 喷射原理

```
执行流程（以 8==D~ 为例）：
  1. 解析：cmd = '8==D', eject = True
  2. 计算：m[ptr] += 2
  3. 判定：如果 eject == True → putchar(m[ptr])
```

### 4.4 字节码标记

```python
eject_bit = 0x80  # 高位标记
bytecode_byte = op | (0x80 if has_eject else 0x00)
```

## 5. 字节码格式

### 5.1 Opcode 定义

```python
OP_MAP = {
    "8{":      0x01,  # 循环开始
    "}D":      0x02,  # 循环结束
    "~":       0x03,  # 输入
    "ADD":     0x04,  # 加法
    "SUB":     0x05,  # 减法
    "MOV_R":   0x06,  # 右移
    "MOV_L":   0x07   # 左移
}

EJECT_BIT = 0x80  # 喷射标记
```

### 5.2 字节码结构

**单字节指令**（无操作数）：
```python
[opcode | eject_bit]
# 例：8{ → 0x01, 8{ + ~ → 0x81
```

**双字节指令**（操作码 + 操作数）：
```python
[opcode | eject_bit, operand]
# 例：8==D~ → 0x84, 0x02
```

### 5.3 虚拟机执行循环

```python
while pc < len(bytecode):
    raw_op = bytecode[pc]
    op = raw_op & 0x7F      # 清除喷射位
    eject = raw_op & 0x80   # 提取喷射位
    
    # 执行指令...
    
    if eject:
        print(chr(mem[ptr]), end='', flush=True)
    
    pc += 1
```

## 7. 示例解析

### 7.1 输出 "73"（BG5CRL 专属签名）

```pink8d
// 设置 '7' (ASCII 55) 并喷射
8=======================================================D~

// 递减 4 变成 '3' (ASCII 51) 并喷射
8....D~

// 右移到新格
8>

// 从键盘读取一个字符
~

// 左移回原位
<8

// 喷射当前格内容
8D~
```

### 7.2 字符串循环输出

```pink8d
// 假设 ptr 指向字符串 "HELLO"
// 循环输出字符串
8{
    8D~    // 喷射当前字符
    8>     // 右移到下一个字符
}D
```

### 7.3 字节码解析示例

**Pink8D 源码**：
```pink8d
8==D~
```

**编译过程**：
1. 词法分析：`['8==D~']`
2. 解析：`cmd = '8==D', eject = True`
3. 计算：`val = 2`（两个 `=`）
4. 生成字节码：`[0x84, 0x02]`
   - `0x04 | 0x80 = 0x84`（ADD + 喷射）
   - `0x02`（操作数）

**虚拟机执行**：
```python
m[ptr] = (m[ptr] + 2) % 256  # 加 2
print(chr(m[ptr]))           # 喷射输出
```
