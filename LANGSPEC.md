## 1. 设计哲学

### 1.1 核心原则
- **形态即功能**：代码的视觉长短直接对应操作的物理强度。
- **非对称美学**：赋值指令是封闭的（```8...D```），位移指令是开放的（```8...>```）。
- **物理类比**：代码模拟液压推进。加压（赋值）、位移（指针）、空腔（循环）与弹射（跳转）构成了物理基础。

## 2. 词法规范

### 2.1 正则表达式 (v3.0 增强型)
```pattern = r'8\[H\d:.*?\]D|8\[=*\]D|(~~|~)?(8[=\.]*[D>\}]|<[=\.]*8|8\*D|8\{)(~~|~)?'```

### 2.2 Token 类型判定
| Token 模式 | 类型 | 示例 | 公式/逻辑 |
|:---|:---|:---|:---|
| ```8[=.]*D~?``` | 赋值指令 | ```8====D~``` | 值 = ```=``` 数量 - ```.``` 数量 |
| ```8[=]*>~?``` | 右移指令 | ```8===>``` | 距离 = ```=``` 数量 + 1 |
| ```<[=]*8~?``` | 左移指令 | ```<===8``` | 距离 = ```=``` 数量 + 1 |
| ```8*D``` | 动态弹射 | ```8*D``` | PC = m[ptr] (自举核心) |
| ```8{``` / ```}D``` | 逻辑空腔 | ```8{ ~ }D``` | while(m[ptr]) { ... } |

## 3. 指令集

### 3.1 基础指令
- **压力赋值**：```8====D``` → ```m[ptr] += n``` (向内挤压)。
- **减压赋值**：```8....D``` → ```m[ptr] -= n``` (向外抽吸)。
- **动力位移**：```8===>``` → ```ptr += n+1``` (动力右喷)。
- **回溯位移**：```<===8``` → ```ptr -= n+1``` (动力左喷)。

### 3.2 自举指令
- **动态弹射 (8*D)**：活塞撞击瞬间，指令指针立即跳转到当前格压力值指向的地址。

## 4. 即时喷射与强力流体

### 4.1 即时喷射 (v2.0 继承)
任何指令加上 ```~``` 后缀，会在动作完成后立即执行 ```putchar(m[ptr])```。
- 示例：```8====D~``` (加压后立即输出字符)。

### 4.2 强力流体 (v3.0 新增)
使用双波浪线 ```~~``` 开启工业级文件流交互：
- ```~~8D```：**强力抽吸**。从当前对准的轨道 (Hatch) 读取 1 字节。
- ```8D~~```：**强力灌注**。向当前对准的轨道 (Hatch) 写入 1 字节。

## 5. 🚀 机体铭牌与轨道协议

### 5.1 机体铭牌 (Machine Plate)
在逻辑区开始前声明轨道咬合关系：
```8[H0:input.8d]D``` // 0号舱门：通常定义为输入轨道  
```8[H1:output.c]D``` // 1号舱门：通常定义为输出轨道  
```---```

### 5.2 舱门切换 (Hatch Alignment)
- ```8[]D```：旋转至 0 号舱门。
- ```8[=]D```：旋转至 1 号舱门。
- 通过切换舱门，程序可以在多个输入/输出轨道间快速分发行程。

## 6. 字节码格式

### 6.1 Opcode 与标记
- ```EJECT_BIT = 0x80``` (高位标记喷射状态)。
- 指令采用 3 字节结构：```[opcode | eject_bit, operand_high, operand_low]```。

### 6.2 虚拟机执行循环
虚拟机解析字节码，执行物理动作。若检测到 ```eject_bit```，则根据当前轨道设置执行输出。

## 7. 内存模型

### 7.1 规格参数
| 参数 | 规格 | 备注 |
|:---|:---|:---|
| 内存大小 | **65,536** | 16-bit 地址空间 |
| 数值范围 | 0-255 | 8-bit 压力值 |
| 溢出处理 | 模运算 | 活塞环绕特性 |

### 7.2 环形缓冲逻辑
```ptr = (ptr + delta) % 65536```  
```m[ptr] = (m[ptr] + delta) % 256```

## 8. 附录

### 8.1 ASCII 速查表
- ```8``` (56)：底座。
- ```7``` (55)：```8=======================================================D```。
- ```*``` (42)：弹射核心。

### 8.2 错误处理
- **循环不匹配**：```8{``` 或 ```}D``` 未配对，抛出 ```SyntaxError```。
- **指针/值溢出**：执行模运算，模拟物理环绕，不报错。
- **EOF 感应**：读取遇到文件尾时，当前格压力强制复位为 ```0```。

|:---|:---|
